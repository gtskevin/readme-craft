# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

- **Do NOT open a public GitHub issue**
- Open a [Security Advisory](https://github.com/gtskevin/repo-showcase/security/advisories/new)
- Or email the maintainer directly

We'll respond within 48 hours and work with you to address the issue.

## Scope

This skill generates markdown and SVG files. It does not execute arbitrary code on user machines. The Python scripts (`generate_badges.py`, `quality_check.py`) are read-only utilities.
